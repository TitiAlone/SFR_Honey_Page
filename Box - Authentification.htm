Mise à jour requise

Merci de mettre à jour votre Box de SFR. Pour effectuer la mise à jour, identifiez-vous en suivant les instructions ci-dessous :

Identification par mot de passe

Saisissez votre identifiant et votre mot de passe puis cliquez sur le bouton Valider.

Aide

Une mise à jour importante est disponible pour votre Box de SFR, identifiez-vous afin d'effectuer cette mise à jour, avant de pouvoir surfer de nouveau.

Identification par mot de passe : Saisissez votre identifiant (par défault, l'identifiant est "admin") et votre mot de passe pour accéder à l'interface d'administration de votre Box.

Le mot de passe est le code WiFi de votre Box, il se trouve par défaut derrière votre Box.

Mise à jour requise

La configuration de votre Box nécessite la mise à jour du firmware. Veuillez s'il vous plait vous identifier afin de réaliser la mise à jour pour pouvoir à nouveau surfer sur le web.

Pour plus d'informations, cliquez sur le bouton "?" en haut de cette page pour de l'aide.
